Detectar idioma de el navegador con javascript

15/7/2020

Este es el código javascript que voy a usar en todas mis weebapps para detectar el idioma de navegador y almacenarlo en una variable.

la funcion detectaridioma funciona como filtro, en este ejemplo el idioma predeterminado es el ingles, en caso de que el idioma detectado no sea español la variable de idioma quedara con el valor de idioma predeterminado, en esta caso el ingles.

HTML:

//variables de idioma
var idioma= "ingles"; //idioma predeterminado
var idiomanavegador= navigator.language;
//filtrar el idioma de el navegador
detectarIdioma();

En caso de que se use un selector de idioma predeterminado simplemente hay que agrear lo siguiente y cambiar NombreProyecto por el nombre que corresponda.

detectaridioma.js:

  // idioma
  //variables de idioma
  idiomanavegador= navigator.language;//idioma del navegador.
 var idioma= "ingles";//idioma predeterminado.
  //si no hay un idioma preseleccionado no pasa nada.
  if (localStorage.NombreProyectoIdiomaSeleccionado == undefined){ detectarIdioma();};
 //si  hay un idioma preseleccionado idoma pasa a ser el idioma preseleccionado.
  if (localStorage.NombreProyectoIdiomaSeleccionado != undefined){ idioma=localStorage.NombreProyectoIdiomaSeleccionado}

    if (idiomanavegador == "es" || idiomanavegador == "es-ES" ) { idioma="español";};
  //español-latinoamerica
    if (idiomanavegador == "es-419" || idiomanavegador == "es-LA"|| idiomanavegador == "es-XL") { idioma="español";};
   
    if (idiomanavegador == "es-AR") { idioma="español";};
