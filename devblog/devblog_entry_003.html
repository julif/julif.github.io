Wordpress custom post type

16/7/2020

Los Custom Post Type (CPT) son un tipo de entrada personalizada que suele utilizarse para separar distinto tipo de contenido.

Los CPT pueden estar en los plugins o en el archivo functions.php, personalmente recomiedo que los CPT siempre sean plugins, de esa manera no se pierden al cambiar el tema de nuestro sitio.

Creando un CPT manualmente

en este ejemplo la funcion libros_cpt crea un nuevo custom post type llamado libros.

          // la funcion libros_cpt
          add_action( 'init', 'libros_cpt' );
          function libros_cpt() {
              // opciones del CTP
              $args = array(
              'public' => true,
              'label' => 'Libros'
            );
            // enlaza la funcion al tema
            register_post_type( 'libros', $args );
          }

en el caso de que se quiera usar el cpt dentro de el tema pero sin llenar de codigo el archivo functions.php, se puede simplemente crear una carpeta en el tema llama custom_post_type donde se pueden alojar los cpt ya sea todos juntos o separados por archivos. Solo tenemos que enlazar esos archivos al archvo functions.php de la siguiente manera:

          include_once "custom_post_type/libros_cpt.php"; // Custom Post:  Libros.
